This is a hello!

And to be honest, there is not much to see here yet. This new system design (which includes the UI) is replacing the previous WordPress backend.

More interestingly, this is now being hosted on a local 6-node Kubernetes cluster (3 physical nodes, 2 virtual machines each). At some point, I intend on re-adding the Kubernetes cluster status page.

As an added bonus, this blog is now being deployed as part of a GitHub Action via a Tailscale connection to the cluster, which builds an image and pings the Kubernetes cluster to re-deploy the blog whenever there is a change.

Posts now get version control, as well as this blog using a far lighter backend (no database or PHP). It is also still sufficiently simple to publish posts compared to WP on my part. What's not to love?
